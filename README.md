# 종가 매매 스크리너 (jongga)

매 거래일 **종가베팅 후보 종목**을 자동으로 추려주는 로컬 도구.
무료 데이터(FinanceDataReader)만 사용하고, 인증·로그인 불필요. 자동주문 없음(후보 추천만).

## ⚠️ 먼저 읽으세요

- 이 도구는 **수익을 보장하지 않습니다.** "확률 게임"인 종가베팅을 규칙대로 빠르게 실행하게 도와줄 뿐입니다.
- 최종 매수 판단·주문은 **본인**이 합니다.
- 실거래 전 반드시 `backtest.py`로 검증하고, 가능하면 **모의계좌로 2주 이상** 연습하세요.

## 구성

| 파일 | 역할 |
|---|---|
| `config.py` | 모든 전략 기준(숫자만 바꾸면 강도 조절) |
| `data.py` | 데이터 수집(전 종목 스냅샷 / 종목별 일봉 / 지수) |
| `strategy.py` | 종가매매 규칙 + 점수화 |
| `screen.py` | **매일 실행** → 후보 TOP N 추천 + 리포트 저장 |
| `backtest.py` | 과거 데이터로 전략 수익성 검증 |

## 전략 규칙 (조사 기반)

종가에 매수 → 다음날 시초가(09:00~09:30) 매도. 후보 조건:

- 거래량 20일 평균 대비 **2배 이상 급증**
- 종가가 당일 고가 대비 **85% 이상**(윗꼬리 짧은 장대 양봉)
- 당일 등락률 **+3~25%** (상한가 근처 과열 제외)
- **5일선 위 / 정배열 / 60일 신고가 근접** 시 가점
- 거래대금 100억+, 시총 500억~5조 (잡주·초대형주 제외)
- 코스피/코스닥 지수 약세일 땐 경고 표시

## 사용법

```bat
pip install -r requirements.txt

REM 오늘의 후보 종목 보기 (오후 3시경 실행 권장)
python screen.py

REM 전략 검증 (유동성 상위 200종목, 2023년~)
python backtest.py 200 2023-01-01
```

리포트는 `reports/` 폴더에 저장됩니다.

## 매일 자동 실행 등록 (Windows 작업 스케줄러)

관리자 PowerShell에서 한 번만:

```powershell
schtasks /create /tn "종가스크리너" /tr "C:\Users\User\claudecode\jongga\run_screen.bat" /sc weekly /d MON,TUE,WED,THU,FRI /st 15:05
```

평일 15:05에 자동 실행되어, 후보 리포트 창이 떠서 알려줍니다. (장 마감 동시호가 직전)

해제: `schtasks /delete /tn "종가스크리너" /f`

## 튜닝

후보가 너무 적으면 `config.py`에서 `MIN_VOL_RATIO`(2.0→1.5), `MIN_CLOSE_POSITION`(0.85→0.8),
`MAX_MARCAP`(5조→10조) 등을 완화하고, 바꿀 때마다 `backtest.py`로 다시 검증하세요.
